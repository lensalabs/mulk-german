## Review-Kriterien für jeden Agent

1. **Padding/Spacing**
   - Konsistente Abstände zwischen Elementen
   - Symmetrie oben/unten/links/rechts
   - Keine Überlappungen
   - Kein unnötiger Leerraum

2. **Funktionalität**
   - Alle Buttons klickbar
   - Alle Links funktionieren
   - Audio/Player funktioniert
   - State-Changes korrekt
   - Keine JavaScript-Errors

3. **Layout-Struktur**
   - Cards füllen verfügbaren Raum
   - Fixed Elements an richtiger Position
   - Scroll-Verhalten korrekt
   - Responsive auf verschiedenen Höhen

4. **Konsistenz**
   - Gleiche Elemente sehen überall gleich aus
   - Gleiche Abstände für gleiche Komponenten
   - Gleiche Interaktions-Patterns
